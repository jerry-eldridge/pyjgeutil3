README_Blender.txt

This folder's 2_Homotopy_Blender.py is a Blender app
(a free 3D modeling software) with a Python API (application
programming interface). This script is run as
Blender > Scripting > Load Script and Run Script and then
switching to the Layout view/panel from Scripting panel.

The script uses two libraries, blender_create_shape.py,
and mini_my_universes.py .
